# Stemming vs lematyzacja

Stemming sprowadza słowa do wspólnej formy rdzeniowej za pomocą reguł heurystycznych, często bez słownika (np. „running” → „run”). Lematyzacja zwraca kanoniczną formę słownikową (lemat), np. „better” → „good”, i zwykle wymaga zasobów językowych. Stemming jest szybszy; lematyzacja dokładniejsza, ale bardziej kosztowna obliczeniowo.
