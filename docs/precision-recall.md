# Precyzja, czułość i F1

Precyzja (precision) mierzy, ile z przewidzianych przypadków pozytywnych jest trafnych: prawdziwe pozytywy podzielone przez wszystkie przewidziane pozytywy. Czułość (recall) mierzy, ile z rzeczywistych przypadków pozytywnych udało się znaleźć: prawdziwe pozytywy podzielone przez wszystkie rzeczywiste pozytywy. Wynik F1 to średnia harmoniczna precyzji i czułości — jedna metryka równoważąca obie, gdy klasy są niezbalansowane.
