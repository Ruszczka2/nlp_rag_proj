# Maszyna wektorów nośnych (SVM)

Maszyna wektorów nośnych (Support Vector Machine) szuka granicy decyzyjnej maksymalizującej margines między klasami w przestrzeni cech. Przy problemach nieliniowych funkcje jądra (np. RBF lub wielomianowa) mapują dane do przestrzeni wyższego wymiaru, gdzie separacja jest łatwiejsza. W klasyfikacji tekstu często stosuje się LinearSVC z rzadkimi cechami TF-IDF, bo jest szybki i dobrze działa na danych wysokowymiarowych.
