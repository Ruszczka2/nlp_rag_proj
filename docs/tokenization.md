# Tokenizacja

Tokenizacja to proces dzielenia tekstu na mniejsze jednostki, takie jak słowa, podwyrazy lub zdania. Zwykle jest to pierwszy krok w pipeline NLP przed czyszczeniem, wektoryzacją lub modelowaniem. Wybór tokenizera wpływa na to, jak obsługiwana jest interpunkcja, skróty i słowa złożone.
