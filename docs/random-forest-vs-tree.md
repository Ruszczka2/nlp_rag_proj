# Las losowy vs pojedyncze drzewo decyzyjne

Pojedyncze drzewo decyzyjne rekurencyjnie dzieli dane regułami opartymi na cechach — jest łatwe do interpretacji, ale podatne na przeuczenie. Las losowy (Random Forest) buduje wiele drzew na losowych podzbiorach danych i cech, a następnie agreguje predykcje głosowaniem lub uśrednianiem. Takie podejście ensemble zwykle lepiej generalizuje i jest stabilniejsze niż jedno drzewo.
