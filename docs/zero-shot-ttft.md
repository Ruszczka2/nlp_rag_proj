# Zero-shot i TTFT

Klasyfikacja zero-shot polega na przypisywaniu etykiet lub kategorii, na których model nie był wprost trenowany, wykorzystując ogólną znajomość języka z pretrainingu. TTFT (Time To First Token) mierzy, jak szybko model językowy zaczyna generować odpowiedź po otrzymaniu promptu. Niski TTFT poprawia doświadczenie użytkownika w aplikacjach interaktywnych, takich jak chatboty i asystenci RAG.
